# Project Context for AI Agents

## Project Overview
This project, `alertwise-cafe-html-theme`, is a static HTML website template designed for a cafe or restaurant. It includes various pages for menu, about us, team, blog, and shop functionalities.

## Tech Stack
- **HTML5**: Structure of the pages.
- **CSS3**: Styling (located in `css/` folder).
- **JavaScript**: Interactivity (located in `js/` folder).
- **jQuery**: Used as a dependency for many plugins.

## Key Directories
- `css/`: Contains stylesheets.
- `js/`: Contains JavaScript files.
    - `custom/`: Custom scripts for the theme.
    - `vendor/`: Third-party libraries and plugins.
- `images/`: Contains static assets.

## Key Files
- `index.html`: The landing page.
- `menu.html`: The menu page.
- `about.html`: About Us page.
- `contacts.html`: Contact page.

## Conventions
- HTML files are in the root directory.
- Assets are organized by type (css, js, images).
