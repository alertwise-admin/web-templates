# Alertwise Cafe HTML Theme

A modern, responsive HTML website template designed for cafes, restaurants, and coffee shops. This template provides a comprehensive set of pages including menu, about us, team, blog, and shop layouts.

## Features

- **Responsive Design**: Looks great on desktops, tablets, and mobile devices.
- **Multiple Page Layouts**: Includes Home, About, Menu, Shop, Blog, and more.
- **Clean Code**: Well-structured HTML, CSS, and JavaScript.
- **Customizable**: Easy to modify styles and content.

## Getting Started

### Prerequisites

You only need a modern web browser to view this template. To edit the code, any code editor (like VS Code, IntelliJ IDEA, or Sublime Text) will work.

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/yourusername/alertwise-cafe-html-theme.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd alertwise-cafe-html-theme
    ```
3.  Open `index.html` in your web browser to view the site.

## Project Structure

- `css/`: Contains all stylesheet files.
- `js/`: Contains JavaScript files (custom scripts and vendor libraries).
- `images/`: Contains images and icons.
- `*.html`: HTML pages for the website.

## Customization

- **Styles**: Edit `css/style.css` or `css/skin.css` to change the look and feel.
- **Scripts**: Custom logic can be found in `js/custom/`.
- **Content**: Modify the HTML files directly to update text and images.
